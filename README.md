## project MONOLITH
 
 **project MONOLITH** is a GUI wraper for various **web-pentesting** and **OSINT** tools.
 #
 ### Stack Used
 - ### frontend
	 - html
	 - css
	 - js
- ### backend
	- python


#

- For running the app in **Linux/Windows** run the installer first.
- Run **setup.bat** in **windows**.
- Run **setup.sh** in **linux**.
